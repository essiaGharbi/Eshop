## Plateforme E-Shop Laravel

E-Shop est une plateforme E-commerce développée avec Laravel, incluant un tableau de bord administrateur et une interface client.
Elle permet la gestion des produits et catégories, le contrôle des utilisateurs, ainsi qu’un système de contact avec les clients.

## Fonctionnalités principales

### Tableau de bord Administrateur

- Gestion des catégories : CRUD complet (Créer, Lire, Modifier, Supprimer)
- Gestion des produits : CRUD complet
- Gestion des utilisateurs : bloquer / débloquer les comptes
- Affichage des commandes

### Interface Client

- Navigation dans les produits et catégories
- Gestion du panier (Cart)
- Système de notes et avis pour les produits (Product Reviews)
- Formulaire Contactez-nous pour envoyer des messages au support

### Système de contact

- Envoi des messages via Mailtrap (SMTP) pour tests
- Confirmation de l’envoi avec message de succès

## Caractéristiques techniques

- Framework : Laravel v12
- Base de données : MySQL
- Authentification : utilisateurs / administrateur
- Interface Dashboard construite avec Blade et Tailwind CSS

## Améliorations futures

- Système de commandes (Orders) avec intégration de paiement en ligne
- Tableau de bord avec statistiques détaillées
- Amélioration de l’interface utilisateur

## Tech Stack

- Laravel v12
- MySQL
- Blade / Tailwind CSS
- Mailtrap (SMTP pour tests)
- Composer et NPM
